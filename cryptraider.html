Crypt Raider

Creating and using C++ components with Unreal blueprints

This project was created as part of a Udemy course on C++ development in Unreal Engine, and a modular level design pack from InfuseStudio.

This projects includes grabbing, moving and box collision components created in C++. Implementation of these components to create a level and puzzle design are my own.
During the creation of this project, I have learnt and practiced the following:

  • how to use pointers and references
  • out parameters
  • creating line trace and collision through C++
  • dependency injection
  • Combining C++ and blueprints
